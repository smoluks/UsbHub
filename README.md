# UsbHub
Simple 4-port hub on FE1.1S

Optional: 
- overcurrent protection on MIC2026
- EEPROM for ID
- 1 common activity LED
- 4 port activity LED

Case printed on 3d printer: https://www.thingiverse.com/thing:4348145
